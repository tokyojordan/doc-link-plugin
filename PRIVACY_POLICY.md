# Privacy Policy for Appian Doc Link Modifier

**Last Updated:** March 11, 2026

## Overview

Appian Doc Link Modifier ("the Extension") is a Chrome extension developed to help users modify Appian documentation URLs. This privacy policy explains how the Extension handles user data.

## Data Collection

The Extension does **NOT** collect, store, or transmit any personal data. Specifically:

- **No data collection**: The Extension does not collect any personal information, browsing history, or usage data.
- **No data transmission**: The Extension does not send any data to external servers or third parties.
- **No data storage**: The Extension does not store any user data locally or remotely.

## Permissions Explanation

The Extension requires the following permissions:

1. **`activeTab`**: Allows the Extension to access the current tab's URL to check if it's an Appian documentation page.
2. **`clipboardWrite`**: Allows the Extension to copy modified URLs to your clipboard.
3. **`scripting`**: Allows the Extension to inject a visual indicator on Appian documentation pages.

All permissions are used solely for the Extension's core functionality and do not involve data collection or transmission.

## How the Extension Works

1. **URL Detection**: The Extension checks if the current webpage is an Appian documentation page (`docs.appian.com`).
2. **URL Modification**: If an Appian docs URL is detected, the Extension modifies version numbers (e.g., `26.2`) to `latest`.
3. **Clipboard Operation**: When you click the copy button, the modified URL is copied to your clipboard.
4. **Visual Indicator**: A small button appears on Appian docs pages for quick access.

All processing happens locally in your browser. No data leaves your device.

## Third-Party Services

The Extension does not use any third-party services, analytics, tracking, or advertising networks.

## Children's Privacy

The Extension does not collect any data from anyone, including children under the age of 13.
